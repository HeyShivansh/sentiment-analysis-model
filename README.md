# Sentiment Analysis Model

A sentiment analysis system that classifies text reviews as **Positive** or **Negative**.
Built using **TF-IDF + Logistic Regression**, with a roadmap to LSTM and BERT.

## Features
- Text preprocessing
- TF-IDF feature extraction
- Logistic Regression classifier
- Model evaluation
- FastAPI deployment (planned)

## Dataset
- Amazon Reviews Polarity Dataset  
- Dataset is not included due to size constraints
